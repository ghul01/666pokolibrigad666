Guild fest

Szabályok:

  • 25 szintű fő várak esetén a feladatokra eső elvárt minimális pontszám 120
  • Minden 15. szintet elért vagy nagyonbb vár esetén elvárt minimális pontszám összesen 1000
  • Az elvégezhető feladatok száma véges. Az utolsó feladat elvégzése 1000 gemért elvárt.
  • Ha valakinek problémája merül fel egy feladattal kapcsolatban, chat-n kérjen segítséget.
  • Ha nem tudsz részt venni, akkor jelezd körlevélben vagy R4-nek lehetőleg a feszt kezdete előtt, hogy a kizárást elkerüljük.
  • Aki a fenti elvárásokat nem teljesíti, és továbbá nem is próbálja erejének megfelelően a lemaradását pótolni, a céhből több napos kizárásra számíthat a jutalomból történő kizárás véget

Beles 05 O levele alapján.

Feladatok:

  • Cargo Ship Trades: hajón kill beválltani az adott mennyiséget ( érdemes fesztre megtartani a familiar[ Totempest: Ship Ahoy ] képességet )
  • Clear a Full LV5 Resource Tile: adott mennyiségű LV5 bányát kell kiszedni ( megszakítani nem lehet )
  • Colosseum Battles: adott mennyiségű csatát kell a colosseumban vívni ( nem kell megnyerni )
  • Colosseum Rank Up: adott helyezést kell előre lépni a colosseumban
  • Complete Admin Quest: admin questet kell nyitni
  • Complete Guild Quest: guild questet kell nyitni
  • Complete Hero Stages: hero pályát kell csniálni ( a már befejezettet is számolja )
  • Complete Phase 3 Hell Events: adott mennyiségű hell eventet kell maxolni
  • Complete Phase 3 Solo Events: adott mennyiségű solo eventet kell maxolni
  • Complete VIP Quests: VIP ládákat kell kinyitni
  • Encounter Labyrinth Guardians: a BOSS-t ( gem manó ) kell előcsalni
  • Forge equipment: adott mennyiségű ruhát kell gyártani ( az upgrade is számít )
  • Gather: bányászás ( FOOD: búza; ORE: érc; WOODS: fa; ROCKS: kő; GOLD: arany; RESOURCES: bármi )
  • Get Dark Essences: adott mennyiségű essencet kell gyűjteni ( ne felejts törölni )
  • Get LV19+ Dark Essences: 19+ os essencet kell gyűjteni
  • Heal Wounded Soldiers: katonákat kell gyógyítani
  • Hit Monsters: szörnyeket kell ütni ( érdemes LV1-es szörnyet ütni, mert ahhoz kell a legkevesebb energy )
  • Incrase Might ( Quest ): mightot kell növelni ( a turf questet számolja csak )
  • Incrase Might ( Research ): mightot kell növelni kutatással
  • Incrase Might ( Troops ): mightot kell növelni katona gyártással
  • Incrase Might Hero: mightot kell növelni hős fejlesztéssel ( két hős liláról aranyra húzása kell hozzá )
  • Incrase Total Might: mightot kell növelni bármivel ( gyógyítás nem számít )
  • Merge Pacts: merge pactot kell gyártani ( érdemes 1a vagy 1b-t, mert ahoz kell a legkevesebb idő )
  • Merge Skillstones: skillstone gyártás
  • Open Mystery Boxes: mystery box nyitás ( az vállalja aki sűrűn fel tud nézni, mert feszt alatt szarul adja )
  • Purchase Special Bundles: speciális csomagokat kell vásárolni pénzért
  • Random Quest: bármilyen feladat ( Lásd alul )
  • Rank In Top70 Hell Event: összesítésben top 70-be kell lenned
  • Rank In Top70 Solo Event: összesítésben top 70-be kell lenned
  • Research Tech: kutatást kell befejezned ( érdemes rövid, kis kutatásokat indítani )
  • Send Help To Your Guildmates: helpekre kell nyomnod
  • Spend Gems: gem költés
  • Spend Guild Coins: guild coin költés
  • Supply: nyersanyag küldés ( FOOD: búza; ORE: érc; WOODS: fa; ROCKS: kő; GOLD: arany; RESOURCES: bármi ) oda-vissza küldözgess valakivel, akkor kevesebb nyersit kell felhasználni
  • Time Reduced Using Speed Up Merging: speedelni ( gyorsítani ) kell merge gyártást
  • Time Reduced Using Speed Ups: speedelni ( gyorsítani ) kell bármit. ( fejlesztés, építés, gyógyítás, fal, stb. )
  • Train Soldiers: katonát kell gyártani ( T1-el jársz a legjobban, ahhoz kell a legkevesebb nyersi és idő )
  • Use Familiar Attack Skills: familiarissal kell támadni
  • Use Fragment: fragmenteket kell elhasználni familiaris fejlesztésre
  • Use Holy Stars: megadott holy-t kell használni
  • Use The Turf Boost: adott termelési boostot kell használni
  • Win Darknest Coalition Battles ( Rally Capitain Only ): általad indított darkot kell nyerni ( az essence nem érdekes )
  • Familiar EXP növelés: familiaris fejlesztése adott ponttal ( csak az exp potiont és az elixirt számolja )
